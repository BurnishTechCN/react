# 监控系统前端

## 运行前端

* docker 方式（推荐）

  - `docker-compose up`

* 手工方式

  - `cd /client`
  - `npm install`
  - `npm start`

* 访问[http://localhost:9000](http://localhost://9000)


## 运行 api

* `git clone git@github.com:BurnishCN/owl.git`
* `docker-compose up`


## 打包

`npm build`


## 单元测试

`npm run coverage`
